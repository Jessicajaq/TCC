Programação à Crianças

Desejando melhorar a flexibilidade de aprendizado e fazer com que mais pessoas se interessem pela área de TI, desde jovens, este projeto tem por objetivo a criação de uma plataforma de ensino a distância com foco em didáticas para crianças e adolescentes com uma fácil usabilidade.

Componentes externos usados no projeto:

O Bootstrap é um framework que contém classes css e arquivos javascript "pré-prontos". Seus arquivos e sua vasta documentação proporcionam uma otimização no desenvolvimento facilitando a criação de um site e a personalização do mesmo.

O Flaticon é um site que possui uma vasta biblioteca de diversos icones. Os icones usados neste projeto são de uso gratuito deste site.

O Freepik é um site com uma grande biblioteca de imagens de alta qualidade. As imagens usadas neste site foram de uso gratuito do freepik.

Este site é um projeto desenvolvido para o trabalho de conclusão de curso para obtenção do título de graduação em Ciência da Computação, apresentado à Universidade Paulista pela aluna Jéssica Mariana Jaqueto de Lima no primeiro semestre de 2021.

Este site é apenas um projeto de estudo. Nenhum link presente aqui possui alguma ligação, patrocinio ou qualquer divulgação financiavel do mesmo.
